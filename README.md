# ICDS Food Safety Training

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**An Interactive Training Platform for ICDS Midday Meal Workers**

A professional React.js-based e-learning application designed to educate Integrated Child Development Services (ICDS) Midday Meal Workers on comprehensive food safety practices. Built with accessibility in mind for field training sessions across diverse literacy levels.

---

## 📋 Overview

The ICDS Food Safety Training application provides structured, visual learning modules covering all essential aspects of food safety in midday meal preparation. This digital training solution is specifically designed for frontline workers, ensuring clear understanding through interactive content and practical guidance.

### Key Features

✅ **Interactive Learning** — Engaging modules with visual aids and practical examples
✅ **Accessible Design** — Intuitive interface for users with varying literacy levels
✅ **Comprehensive Coverage** — All five pillars of food safety addressed
✅ **Flexible Navigation** — Self-paced or instructor-led training modes
✅ **Responsive Design** — Works seamlessly across devices

---

## 📚 Training Modules

### 1. Personal Hygiene
- Proper handwashing techniques and frequency
- Required attire and personal protective equipment
- Health protocols and illness reporting for food handlers
- Personal cleanliness standards

### 2. Workplace Hygiene
- Kitchen sanitation and cleaning schedules
- Waste disposal management and best practices
- Pest control and prevention measures
- Equipment cleaning and maintenance

### 3. Proper Cooking
- Temperature control and monitoring
- Safe cooking practices and techniques
- Cross-contamination prevention strategies
- Food preparation workflows

### 4. Proper Storage
- Raw material storage guidelines
- First-In-First-Out (FIFO) inventory practices
- Temperature-sensitive food handling
- Storage facility requirements

### 5. Nutrition
- Balanced meal planning principles
- Portion size guidelines and standards
- Micronutrient awareness and requirements
- Nutritional value preservation

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed on your development machine:

- **Node.js** — Version 16 or higher
  - Download: [https://nodejs.org/](https://nodejs.org/)
- **npm** — Comes bundled with Node.js (or **yarn** as alternative)

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repo-url>
   ```

2. **Navigate to project directory:**
   ```bash
   cd icds-food-safety-training
   ```

3. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

4. **Start the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```

5. **Access the application:**
   Open your browser and navigate to [http://localhost:3000](http://localhost:3000)
