# ICDS Food Safety Training

A React.js-based interactive presentation for training ICDS (Integrated Child Development Services) Midday Meal Workers on food safety practices, designed for clarity and ease of use in field training sessions.

## Overview

This training module delivers structured, visual guidance across the core pillars of food safety in midday meal preparation, designed for accessibility by frontline workers with varying literacy levels.

## Modules Covered

- **Personal Hygiene** — handwashing, attire, health protocols for food handlers
- **Workplace Hygiene** — kitchen sanitation, waste disposal, pest control
- **Proper Cooking** — temperature control, cooking practices, cross-contamination prevention
- **Proper Storage** — raw material storage, FIFO practices, temperature-sensitive handling
- **Nutrition** — balanced meal planning, portion guidelines, micronutrient awareness

## Tech Stack

- React.js
- HTML5 / CSS3

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
git clone <repo-url>
cd icds-food-safety-training
npm install
npm start
```

The app will run at `http://localhost:3000`.

## Project Structure

src/
├── components/       # Reusable UI components
├── modules/          # Training modules (hygiene, cooking, storage, nutrition)
├── assets/           # Images, icons, media
└── App.js

## Usage

Navigate through modules sequentially or jump to a specific topic via the module menu. Designed for both self-paced learning and instructor-led sessions.
